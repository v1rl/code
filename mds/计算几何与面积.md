### 1) 题目归类：它的本质是什么题？

**核心类型：解析几何 / 计算几何（向量叉积/行列式求面积）**
本质上是把“**三角形面积固定**”转成“**有向面积（叉积）等式**”，由于点 (O) 被限制在 **x 轴**（只有一个未知数 (x)），所以叉积表达式对 (x) 是**一次函数**，最终变成“解一个带绝对值的一次方程 + 特判”。

你以后看到类似关键词，可以立刻联想到同一套路：

- “给定两点，找第三点在某条直线/坐标轴上，使面积为常数”
- “面积=常数、共线、平行、垂直”
  → 优先想 **点积/叉积**（点积管角度，叉积管面积与共线）

这题还可以归到一个更“模型化”的类里：

> **“限制点在一条直线上 ⇒ 面积约束 ⇒ 叉积变线性方程”**
> —— 一维未知 + 二维几何约束 的典型模板题。

### 2) 假如没有题解，我该如何从零推导？（思维复盘）

我用“自问自答”的方式，把关键抓手怎么出现、怎么转化讲清楚。

#### Step A：先把“要求”翻译成数学量

**问：题目要什么？**
要找 ($O(x,0)$)，使 ($\triangle ABO) 面积 (=2)$。

**问：面积在坐标里怎么写最干净？**
坐标几何里最稳的就是**叉积/行列式**：

$$2S = \left| (B-A) \times (O-A) \right|$$

因为叉积的绝对值就是平行四边形面积，三角形面积乘 2 刚好对应它。

所以目标直接变成：

$$\left| (B-A) \times (O-A) \right| = 4$$

这里的“4”就是你代码里的常数来源：**面积 2 ⇒ 两倍面积 4**。

------

#### Step B：把未知 (x) 代进去，看它变成什么“代数结构”

设
$$
(A(x_1,y_1)), (B(x_2,y_2)), (O(x,0))
$$

$$
B-A = (x_2-x_1,; y_2-y_1),\quad O-A=(x-x_1,; -y_1)
$$

叉积（二维向量叉积标量）：
$$
(B-A)\times(O-A)=(x_2-x_1)(-y_1)-(y_2-y_1)(x-x_1)
$$
你看到了关键现象：

- 这是 **关于 (x)** 的**一次式**（线性函数）。
- 所以本题不是“搜索/二分/三分”，而是直接“解一次方程 + 绝对值”。

------

#### Step C：绝对值方程怎么处理？先看会不会退化

**问：什么时候会“不能除”？**
当$ (y_2-y_1=0)$（即$ (y_1=y_2)$）时，上式里 (x) 的系数没了，叉积对 (x) **不再变化**。

这就是你 AC 代码里的关键特判：`if (y1 == y2)`。

- 若 (y_1=y_2)，叉积化简为：
  $$
  (x_2-x_1)(-y_1)
  $$
  它**与 (x) 无关**，所以要么：

  - 本来面积就等于 2 ⇒ **任意 x 都行**（输出 0 就行）
  - 否则 ⇒ **无解**（输出 no answer）

这一步是“从数学结构里挖特判”，非常典型：

> **一旦你准备“除以某个量”，先问它会不会为 0；为 0 时意味着什么几何退化？**

------

#### Step D：非退化时直接解线性方程（选一个符号即可）

当 $(y_1\neq y_2)$ 时：
$$
\left|(x_2-x_1)(-y_1)-(y_2-y_1)(x-x_1)\right|=4
$$
绝对值 ⇒ 两种有向面积：
$$
(x_2-x_1)(-y_1)-(y_2-y_1)(x-x_1)=4
$$
或
$$
(x_2-x_1)(-y_1)-(y_2-y_1)(x-x_1)=-4
$$
解出来就是：
$$
x = x_1 + \frac{4-(x_1-x_2)y_1}{y_1-y_2} \quad (\text{取其中一个符号即可})
$$
这就是你第一份代码的：

```cpp
double ans = (4 - dx * y1) / dy + x1;
```

其中 `dx=x1-x2, dy=y1-y2`。

------

#### Step E：理解你第二份代码“为什么也对”（更像几何推导）

你第二份写法：

```cpp
double x = x1 + (0 - y1) / (y2 - y1) * (x2 - x1); // 直线AB与x轴交点的x坐标
x += 4.0 / (y1 - y2);
```

第一段算的是：**直线 AB 与 x 轴的交点 (X_0)**。

然后 `+ 4/(y1-y2)` 相当于：
在 x 轴上把点从交点平移一段，使得有向面积从 0 变为 ±4。

这个视角非常值钱：

> **把“面积”看成“某个线性函数”，交点处有向面积为 0，往左右移动面积按固定斜率变化。**

这能帮你以后更快“看出来是一次式”。

### 3) 从这题提炼一个通用解题方法论（遇到陌生题不空白）

给你一套我建议你训练成“下意识流程”的框架，专治“不会从哪下手”。

#### 方法论：把题目强行拆成 4 个固定问题

##### (1) 未知量是谁？维度是多少？

- 这题未知是 (x)（1 维）。
- 一维未知很大概率 ⇒ 最终是 **方程 / 单调二分 / 枚举少量情况**。

##### (2) 题目约束属于哪类“几何/代数不变量”？

- 面积、共线、平行、垂直、距离
  分别对应：
  - 面积/共线 ⇒ **叉积/行列式**
  - 垂直/投影 ⇒ **点积**
  - 距离/圆 ⇒ **平方距离**
  - 直线 ⇒ **斜率/一般式/参数式**

看到“面积=常数”，第一反应就应该是：**叉积**。

##### (3) 代入限制，把表达式变成“你熟悉的代数结构”

这里点 (O) 在 x 轴 ⇒ (O(x,0))。
一代入，叉积自动变成一次函数 ⇒ 你就知道“解一次 + 特判”。

> 训练重点：别急着算，先观察“对未知量是几次”。

##### (4) 特判从哪里来？

只问一句话就够：

- “我接下来要不要除以某个量？它可能为 0 吗？为 0 表示什么退化？”
  这题就是 (y_1-y_2=0)。

把这四步练熟，你遇到陌生题就不会“脑中一片空白”，因为你永远有固定抓手去推进。

### 4) 顶尖选手视角：从思路到代码的“编码流”

这里给你一个很“比赛化”的写码顺序（你说你常写着写着卡住，这个流程能显著降低卡顿）：

1. **先写输入输出壳子**（solve、读四个数）
2. **先写特判**（所有会导致除 0 / 退化的情况）
   - `if (y1 == y2)`：
     - `if (abs(1LL*y1*(x1-x2)) == 4) print 0`
     - else print `"no answer"`
     - return
3. **主公式只保留一条**（别在代码里推导）
   - `long double x = x1 + (4 - (x1-x2)*y1) / (y1-y2)`
4. **输出精度**
   - 题目允许 1e-3，直接 `fixed<<setprecision(10)` 稳。
5. **小心类型**
   - 用 `long long` 承接乘法，避免 int 边界心里负担（虽然本题 int 也够，但比赛里养成习惯更好）。

你已经写对了；你下一步要练的是：**把“推导”放在纸上/脑内一次性完成，代码里只放最终公式 + 特判结构**。顶尖选手写码快，核心原因就是“代码不承载推导过程”。

### 额外值得吸收的一点（可选但很有用）

#### 1）“有向面积”比“面积”更好用

你代码里其实已经在用有向面积了：先构造一个解（选定符号），输出即可。
很多几何题都会出现绝对值，你要习惯的策略是：

- **先去掉绝对值做“有向版本”**
- 得到解后，如果题目只要“存在一个解”，通常随便取一个符号就够了
- 只有当题目要“最小/最大/计数”时，才需要系统处理两个分支

#### 2）“点被限制在直线/坐标轴上”通常意味着“参数化降维”

这题是最经典的降维：二维点 (O) 被限制到一维集合（x 轴），于是几何约束直接变代数一次式。
以后你看到“点在某条线段/直线/圆上”，第一想法就应该是：

- 用参数表示它（直线：$(P=P_0+t\vec v)$，圆：$((x,y)=(x_0+r\cos\theta,y_0+r\sin\theta))$)
- 再把题目条件代进去看结构
