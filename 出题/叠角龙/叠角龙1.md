### 题目背景

“随我一同冲入毁灭吧！”

在《英雄联盟》的团战中，暗裔剑魔·亚托克斯开启了终极技能“大灭”（World Ender），准备迎接一场杀戮盛宴。大灭状态下，剑魔可以通过连续不断的攻击来汲取敌人的生命力并叠加自身的鲜血魔井层数。但大灭的持续时间是有限的，你需要精准规划他的进场与输出时机。

### 题目描述

敌方共有 $n$ 个单位站成一排，依次编号为 $1$ 到 $n$。击杀第 $i$ 个单位可以为剑魔叠加 $a_i$ 层鲜血魔井（注意：部分敌人身上带有重伤或反甲效果，强行击杀会导致层数受损，即 $a_i < 0$）。

剑魔需要选择一个**连续的区间**发起突进并一路斩杀。受到“大灭”持续时间的约束，剑魔**最多只能连续斩杀 $k$ 个单位**。同时，为了保证大灭不被浪费，剑魔**至少需要斩杀 1 个单位**。

请问，在保证连续斩杀数量不超过 $k$ 的前提下，剑魔最多可以获得多少层鲜血魔井？

### 输入格式

第一行包含两个整数 $n$ 和 $k$，分别表示敌方单位的总数，以及大灭状态最多允许连续斩杀的单位数量。

第二行包含 $n$ 个整数，第 $i$ 个整数代表 $a_i$，即击杀第 $i$ 个单位所获得的层数。

### 输出格式

输出一个整数，表示剑魔能获取的最大连续累加层数。

------

### 输入输出样例

**输入样例 1：**

Plaintext

```
6 3
1 -3 5 1 -2 3
```

**输出样例 1：**

Plaintext

```
6
```

> **样例 1 解释：**
>
> $k = 3$，意味着最多只能连续选 3 个单位。
>
> 最优策略是选择编号 3 和 4 的单位，获得 $5 + 1 = 6$ 层。虽然把编号 6 加进来总和可能变大，但中间隔了负数，且如果强行连起来长度会超过 $k=3$ 的限制。注意这里实际选取的长度为 2，没有超过上限。

**输入样例 2：**

Plaintext

```
4 2
-5 -2 -9 -1
```

**输出样例 2：**

Plaintext

```
-1
```

> **样例 2 解释：**
>
> 全局都是负数，且规定至少要斩杀 1 个单位。此时为了让损失最小化，只能选择受损最少的第 4 个单位（-1 层），输出 -1。

------

### 数据范围

- $1 \le k \le n \le 10^5$
- $-10^4 \le a_i \le 10^4$
