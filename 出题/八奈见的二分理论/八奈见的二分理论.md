八奈见的二分理论



八奈见的二分理论将人分为了“幼驯染”和“泥棒猫”两种类型，但经过多年研究，你惊奇地发现：有些人可能同时拥有这两种属性！

现在，我们有分别对应两种属性的两份数字化名单，你想要据此得到一份同时具有两种属性的人的名单。



第一行输入空格隔开的两个正整数 $n, m$，分别代表第一份名单和第二份名单上的人数 $(1 \leq n, m \leq 10^5)$。

第二行输入空格隔开的 $n$ 个正整数，代表第一份名单中的人，第 $i$ 个正整数 $a_i$ 满足 $(1 \leq a_i \leq 10^9)$。

第三行输入空格隔开的 $m$ 个正整数，代表第二份名单中的人，第 $i$ 个正整数 $b_i$ 满足 $(1 \leq b_i \leq 10^9)$。



输出一行：按照数字从小到大，输出目标名单，相邻数用空格隔开。



对于20%的数据 $(1 \leq n, m \leq 20)$ ，$(1 \leq a_i, b_i \leq 100)$

对于60%的数据$(1 \leq n, m \leq 10^3)$，$(1 \leq a_i, b_i \leq 10^5)$

对于100%的数据$(1 \leq n, m \leq 10^5)$，$(1 \leq a_i, b_i \leq 10^9)$
