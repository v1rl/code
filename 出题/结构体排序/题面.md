## 题目名称：捍卫者的荣光

### 题目背景

“准备好迎接震撼了吗？我已经准备好了。”

在诸王峡谷的枪林弹雨中，又一场残酷的 Apex 比赛落下了帷幕。现在，系统需要根据各个传奇（选手）在整场比赛中的综合表现，生成一张最具权威性的 MVP 结算排行榜。

身为顶尖黑客“密客”（Crypto），你黑入了比赛的后台，决定亲手编写这段生成排行榜的排序算法。

### 题目描述

本次比赛共有 $n$ 名选手参赛。系统记录了每位选手的四项关键信息：

1. **传奇代号** $S$（一段仅由大小写英文字母组成的字符串）。
2. **击杀数** $K$（即总淘汰人数）。
3. **造成伤害** $D$（即全场打出的总伤害量）。
4. **存活时间** $T$（单位为秒）。

为了绝对的公平，排行榜需要严格按照以下优先级进行排序：

1. **优先按击杀数** $K$ 从高到低排序，击杀越多排名越靠前；
2. 如果击杀数相同，则按**造成伤害** $D$ 从高到低排序；
3. 如果击杀数和造成伤害都相同，则按**存活时间** $T$ 从长到短排序；
4. 如果以上三项战斗数据**完全一致**，则按**传奇代号** $S$ 的**字典序**从前到后（即 A 到 Z 的升序）进行排列，以示对字母表顺序的尊重。

请你输出按照上述规则排序后的最终选手排名表（只需输出排好序的选手代号即可）。

### 输入格式

第一行包含一个正整数 $n$，表示参赛选手的总人数。

接下来 $n$ 行，每行包含四个数据，依次为选手代号 $S$、击杀数 $K$、造成伤害 $D$ 和存活时间 $T$。数据之间用一个空格隔开。

### 输出格式

输出 $n$ 行，每行一个字符串，表示按规则排序后从第一名到最后一名的传奇代号。

------

### 输入输出样例

**输入样例 1：**

Plaintext

```
5
Wraith 12 2500 1200
Bloodhound 10 2800 1350
Octane 12 2500 1100
Lifeline 10 2000 1350
Pathfinder 12 2500 1200
```

**输出样例 1：**

Plaintext

```
Pathfinder
Wraith
Octane
Bloodhound
Lifeline
```

> **样例 1 解释：**
>
> - `Pathfinder`、`Wraith`、`Octane` 的击杀数最高（均为 12），因此排在前三。
> - 在这三人中，三人的伤害同为 2500。
> - 比较存活时间，`Pathfinder` 和 `Wraith` 均为 1200 秒，大于 `Octane` 的 1100 秒，因此 `Octane` 锁定第三。
> - `Pathfinder` 和 `Wraith` 的战斗数据完全一致，比较字典序，`P` 在 `W` 前面，因此 `Pathfinder` 第一，`Wraith` 第二。
> - 剩下的两人中，`Bloodhound`（伤害 2800）排在 `Lifeline`（伤害 2000）之前。

------

### 数据范围

- $1 \le n \le 10^5$
- 选手代号 $S$ 的长度不超过 15，且仅包含大小写英文字母。
- $0 \le K \le 100$
- $0 \le D \le 9999$
- $0 \le T \le 3000$

### 给命题人的温馨提示（含标准解答）

这道题的数据范围 $n = 10^5$，这意味着诸如冒泡排序、选择排序等 $O(n^2)$ 的算法将会直接超时（TLE）。选手必须使用 $O(n \log n)$ 复杂度的排序算法，在 C++ 中最优雅的方式就是重载 `operator<` 或者手写 `cmp` 函数并调用 `std::sort`。

以下是我为你准备的本题标程（Standard Program）：

C++

```
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

// 定义选手结构体
struct Player {
    string name;
    int kills;
    int damage;
    int time;
};

// 核心排序逻辑 (返回 true 表示 a 应该排在 b 的前面)
bool cmp(const Player& a, const Player& b) {
    if (a.kills != b.kills) {
        return a.kills > b.kills;       // 1. 击杀高者优先 (降序)
    }
    if (a.damage != b.damage) {
        return a.damage > b.damage;     // 2. 伤害高者优先 (降序)
    }
    if (a.time != b.time) {
        return a.time > b.time;         // 3. 存活久者优先 (降序)
    }
    return a.name < b.name;             // 4. 字典序小者优先 (升序)
}

void solve() {
    int n;
    if (!(cin >> n)) return;
    
    vector<Player> players(n);
    for (int i = 0; i < n; i++) {
        cin >> players[i].name >> players[i].kills >> players[i].damage >> players[i].time;
    }

    // 调用 C++ STL 标准排序算法 (平均复杂度 O(N log N))
    sort(players.begin(), players.end(), cmp);

    // 输出排行榜
    for (int i = 0; i < n; i++) {
        cout << players[i].name << "\n";
    }
}

int main() {
    // 优化输入输出流速度，防止大数据读取超时
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    solve();
    return 0;
}
```
